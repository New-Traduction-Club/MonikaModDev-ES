# Monika After Story (MAS)
Monika After Story es un mod para el juego gratuito [Doki Doki Literature Club](https://www.ddlc.moe) de [Team Salvato](http://teamsalvato.com/). ¡MAS toma como base el Acto 3 para crear un simulador de tu vida eterna con Monika, incluyendo nuevos eventos, mecánicas y metacomentarios!

MAS ha sido traducido al español por Traduction Club!

Esta traducción no es oficial, si deseas descargar el mod original en inglés puedes hacerlo desde está [página](http://www.monikaafterstory.com/releases.html).

Por favor, revisa la página de [Lanzamientos](https://papuweb.wixsite.com/monikaafterstoryesp/descargas) para obtener la versión traducida, estable y más reciente.

### Instalación

1. Navega a la [página de lanzamientos](https://papuweb.wixsite.com/monikaafterstoryesp/descargas).

2. Haz clic en el enlace del **Zip** deseado. Esto descargará un fichero zip a tu ordenador.

3. Extrae el contenido del fichero zip en el directorio raíz, verás un archivo `LEEME.txt`  y una carpeta.

4. Sigue las indicaciones en el archivo `LEEME.txt`.

5. Al ejecutar `Monika_After_Story.exe`/`Monika_After_Story.sh`, se cargará el mod de Monika After Story.

Para más ayuda con la instalación, puedes unirte a [nuestro servidor de Discord](https://discord.gg/usKs37Sh5S).

### Características

* ¡Pasa la eternidad con Monika!
* Docenas de nuevos temas de conversación.
* Ahora puedes hablar con Monika para decirle de qué te gustaría charlar.

## MASL (Android) y Traducción al Español

Además de la versión principal, contamos con **MASL**, una versión adaptada para Android que está disponible en la [Play Store](https://play.google.com/store/apps/details?id=com.z.mas.portby.just6889) y que posee su propio [repositorio](https://github.com/New-Traduction-Club/MonikaAfterStory-Android-port). 

Para gestionar la localización, hemos creado un [repositorio](https://github.com/New-Traduction-Club/MonikaModTL-ES) dedicado exclusivamente a los archivos de traducción. Sin embargo, todo el código fuente del proyecto en español se subirá y mantendrá en este repositorio principal (**MonikaModDev-ES**).
